# 🌍 多语言功能说明 / Multilingual Support

## 🎯 功能概述 / Overview

PNG to JPG 转换器现已支持多语言界面，提供地道的简体中文和英语版本。

The PNG to JPG Converter now supports multilingual interface with native Simplified Chinese and English versions.

## 🔧 使用方法 / How to Use

### 🖱️ 手动切换语言 / Manual Language Switch
1. 在页面右上角找到语言切换按钮
   Find the language switch buttons in the top-right corner
2. 点击 "🇨🇳 中文" 切换到中文
   Click "🇨🇳 中文" to switch to Chinese
3. 点击 "🇺🇸 English" 切换到英语
   Click "🇺🇸 English" to switch to English

### 🤖 自动语言检测 / Auto Language Detection
- 首次访问时自动检测浏览器语言
  Automatically detects browser language on first visit
- 中文浏览器环境默认显示中文界面
  Chinese browser environment defaults to Chinese interface
- 其他语言环境默认显示英语界面
  Other languages default to English interface

### 💾 语言设置记忆 / Language Setting Memory
- 选择的语言会自动保存到浏览器本地存储
  Selected language is automatically saved to browser local storage
- 下次访问时会记住你的语言选择
  Your language choice will be remembered on next visit
